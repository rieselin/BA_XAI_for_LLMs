📐 How SHAP sampling works

SHAP doesn't test every possible word combination. Instead it runs your prompt many times with different words masked, then uses the changes in model confidence to estimate each word's contribution.

  • 64 samples — fast (~2–3 min), rough estimates
  • 128 samples — balanced (~5–7 min), reliable
  • 256 samples — slow (~10–15 min), more stable
  • 512+ samples — very slow, publication-quality
