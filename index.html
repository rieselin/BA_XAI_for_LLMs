Bevor Sie fortfahren

Dieses Tool sendet Ihren Text zur Verarbeitung an einen lokalen KI-Server. Bitte lesen und bestätigen Sie Folgendes, bevor Sie es verwenden.

  • Geben Sie keine personenbezogenen Daten ein (Namen, Adressen, Ausweisnummern usw.)
  • Geben Sie keine vertraulichen, firmeneigenen oder sensiblen Informationen ein
  • Geben Sie keine Gesundheits-, Finanz- oder Rechtsinformationen ein
  • Verwenden Sie nur fiktive, anonymisierte oder öffentlich verfügbare Inhalte

Alle von Ihnen eingegebenen Texte werden vom verbundenen Server verarbeitet. Nach Ende der Sitzung werden keine Daten gespeichert.

Dieser Hinweis erscheint nur einmal pro Sitzung.

Die meisten KI-Modelle sind Black Boxes. Dieses nicht. Probieren Sie drei Methoden aus, die das Modell öffnen und zeigen, was wirklich darin vorgeht.

Chain-of-Thought

Zwingt das LLM, seinen Denkprozess Schritt für Schritt darzulegen, bevor es eine Antwort gibt – wie das „Zeigen der Rechnung" bei einer Matheaufgabe.

SHAP-Attribution

Jedes Wort Ihres Prompts erhält einen Score, der zeigt, wie stark es die Antwort beeinflusst hat. Orange = hat die Antwort vorangetrieben. Blau = hat sie gebremst.

Konfidenz-Score

Sehen Sie, wie sicher das Modell bei jedem Token war. Niedrige Konfidenz weist oft auf unsichere Fakten oder potenzielle Halluzinationen hin.

SHAP-Stichproben
Anzahl der Maskierungsdurchläufe zur Schätzung des Einflusses jedes Wortes.

So funktioniert SHAP-Sampling

SHAP testet nicht jede mögliche Wortkombination. Stattdessen führt es Ihren Prompt viele Male mit unterschiedlich maskierten Wörtern aus und nutzt die Änderungen in der Modellkonfidenz, um den Beitrag jedes Wortes zu schätzen.

  • 64 Stichproben — schnell (~2–3 Min.), grobe Schätzungen
  • 128 Stichproben — ausgewogen (~5–7 Min.), zuverlässig
  • 256 Stichproben — langsam (~10–15 Min.), stabiler
  • 512+ Stichproben — sehr langsam, publikationsreife Qualität
